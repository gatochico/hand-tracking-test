# Somática del Ruido Digital

## Instalación

```bash
npm install
npm run dev
```

## Concepto

La proyección funciona como un mapa vivo de los residuos emocionales y sociales que circulan en internet, pero también de las lógicas algorítmicas que los ordenan, amplifican y solidifican. Cada imagen es un nodo de violencia latente cuya existencia no es neutral: es el resultado de sistemas de clasificación automatizada que, en su aparente objetividad, reproducen desigualdades y trazan nuevas formas de segregación digital —un proceso cercano a lo que se ha llamado technological redlining [[1](https://www.jstor.org/stable/j.ctt1pwt9w5)].

Al acercarse el espectador, cada nodo late, respira y se expande, revelando que la violencia digital no existe en abstracto: se materializa cuando la mirada humana y la arquitectura algorítmica coinciden. La obra pone en evidencia cómo la atención, el cuerpo y los sistemas de datos participan conjuntamente en la activación del daño; cómo el odio, aparentemente disperso, adopta ciclos, ritmos y mutaciones que dependen tanto de quienes lo observan como de los mecanismos automatizados que lo gestionan, lo priorizan y lo reproducen.

La instalación convierte estos procesos invisibles en organismos vivos, permitiendo ver cómo la violencia digital es modulada simultáneamente por nuestras interacciones y por los valores, sesgos y jerarquías inscritas en las infraestructuras tecnológicas contemporáneas.
